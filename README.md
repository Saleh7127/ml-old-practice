## Usage
1. Clone the repository:
```bash
git clone https://github.com/Saleh7127/ml-old-practice.git
cd ml-old-practice
```
2. Open Jupyter Notebook:
```bash
jupyter notebook
```
3. Navigate to the desired notebook and start exploring!
